## 各种文件的用途

tftpd.cpp就是作为tftp服务器一方的程序，没有实现daemon，也就是脱离控制终端的功能

tftp_c.cpp就是所谓tftp发起方的程序。

talk.cpp是服务器一方专门用来交流用的程序

download.cpp就是实际上交流的过程的程序

create_dgram.cpp生成报文的程序
